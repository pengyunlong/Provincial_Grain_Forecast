本文不再作为“联合聚类实验计划”，而是作为 `combined_clustering` 四域联合聚类的细节观察笔记。重点记录：

1. 四域联合聚类实际使用的方法。
2. 人口、经济、口粮、饲料粮四个域分别使用的数据特征。
3. 每个域的项目内原始数据出处、原始字段和原始数据特征情况。
4. 四域等权和对照实验的计算方式。
5. 当前主实验 `M0` 的 5 个簇、省份组成、核心中心特征和稳定性结果。

本文记录的是项目内脚本、CSV 输出和本地数据文件可追溯的信息。除土地面积表已在项目中记录公开来源外，不额外追溯 CSV 上游年鉴或数据库。

## 3. 通用方法

联合聚类的实际流程如下：

```text
读取人口、经济、口粮、饲料粮和土地面积 CSV
筛选 2015-2024 年
按土地面积表统一 31 省行政区划代码
构造人口密度、城镇化率、老龄人口占比、少儿人口占比
对人口密度、人均可支配收入、人均消费支出执行 log1p 转换
按省份构造 20 个核心特征
逐列 z-score 标准化
按实验规则进行域权重缩放或直接拼接
固定 K=5 执行 KMeans
将 C1-C5 的簇编号与 M0 做最优匹配
输出主实验结果、对照实验质量指标、稳定性分析和地图
```

### 3.1 序列特征构造

联合聚类不完整复用四个独立聚类的全部特征，而是只保留“当前水平、长期趋势、必要波动”三类信息。

项目内通用函数 `build_series_features()` 会先按省份和年份排序，再生成：

| 特征 | 含义 |
| :--- | :--- |
| `最新年值` | 2024 年值 |
| `线性趋势斜率` | 2015-2024 年年度序列的一元线性拟合斜率 |
| `变异系数` | 标准差 / 均值，仅口粮域和饲料粮域使用 |

人口域和经济域只取 `最新年值`、`线性趋势斜率`；口粮域和饲料粮域还取 `变异系数`，用于观察消费波动。

### 3.2 `log1p` 转换

人口密度、人均可支配收入、人均消费支出在提取序列特征前先执行：

```text
log1p(x) = ln(1 + x)
```

用途：

1. 降低人口密度、收入、支出跨省绝对差异过大对距离的影响。
2. 让收入和支出的趋势斜率更接近相对增长速度，而不是只反映绝对金额增长。
3. 仍保留高密度、高收入地区与低密度、低收入地区的相对差异。

结果解释时同时保留原始口径字段，例如人口密度、人均收入、人均消费支出、口粮和饲料粮 2024 年原始值。

### 3.3 标准化与域等权

联合聚类先对 20 个核心特征逐列做 z-score 标准化：

```text
z = (x - mean) / std
```

主实验 `M0` 为四域等权。由于各域特征数不同，直接拼接会让特征更多的人口域天然占更大权重。脚本在逐列标准化后，对每个域再乘以：

```text
1 / sqrt(该域特征数)
```

| 指标域 | 特征数 | M0 域内缩放系数 | 预期权重 |
| :--- | ---: | :--- | :--- |
| 人口域 | 10 | `1 / sqrt(10)` | 约 25% |
| 经济域 | 4 | `1 / sqrt(4)` | 约 25% |
| 口粮域 | 3 | `1 / sqrt(3)` | 约 25% |
| 饲料粮域 | 3 | `1 / sqrt(3)` | 约 25% |

`C1` 不做域权重缩放，仅使用逐列 z-score 后的所有特征直接聚类。`C2-C5` 每次移除一个域，并将剩余三个域重新等权。

### 3.4 KMeans 参数

| 参数 | 当前设置 |
| :--- | :--- |
| 聚类算法 | KMeans |
| 距离目标 | 最小化标准化加权特征空间中的类内平方距离和 |
| `n_clusters` | 5 |
| `random_state` | 42 |
| `n_init` | 80 |
| `max_iter` | 300 |
| 空簇处理 | 若某次迭代出现空簇，用距离最近中心最远的样本重置该簇中心 |

### 8.2 饲料粮折算方法

上游脚本：

```text
数据提取与可视化/七类动物性消费/饲料粮/code/计算饲料粮.py
```

计算公式：

```text
饲料粮折算值 =
猪肉 x 3.3 + 牛肉 x 2.0 + 羊肉 x 1.0 + 禽肉 x 1.9
+ 水产品 x 0.8 + 蛋类 x 2.0 + 奶类 x 0.5
```

只有七类动物性消费分项全部非缺失时才计算合计值，否则 `饲料粮折算值（千克/人）` 记为缺失。

## 15. 后续观察问题清单

1. 西藏单省簇是否主要由极低人口密度和口粮趋势斜率共同驱动，需要结合原始口粮年度序列检查。
2. 京津沪簇是否稳定代表“高密度高收入低口粮”类型，可结合 C1-C5 稳定性验证。
3. `cluster=0` 与 `cluster=3` 都有较强饲料粮上升特征，需进一步比较收入、饲料粮最新水平和口粮趋势。
4. `cluster=2` 省份最多，内部差异可能较大，应结合标准化中心热图和 PCA 图观察是否存在可解释的子结构。
5. C1 与 M0 的 ARI 较低，说明不做域等权会明显改变结果；解释 M0 时需要强调四域等权是业务约束。
6. C3、C5 与 M0 的 ARI 较低，说明经济域和饲料粮域对当前联合聚类边界影响较明显。
7. 完全稳定省份可作为各簇核心样本，发生变化次数较高的省份应作为边界样本单独观察。
8. 土地面积为 1996 年静态面积，只作为人口密度分母；不要解释为 2024 年动态面积。

分析与结论：

  1. 权重校正机制的必要性 (C1 对照)：未校正的特征直接等权（C1）与 M0 的 ARI 仅为 0.359。表明人口域庞大的特征数极大地扭曲了分类结构，证明校正域内权重是维持四域同等话语权的基石。
  2. 最核心的区别因子 (C3 & C5 消融)：移除经济域 (C3, ARI=0.409) 或饲料粮域 (C5, ARI=0.395) 会使分类格局发生颠覆性重组，表明「经济发展度」与「饲料粮需求及变化率」是驱动省份间异质性的最关键源动力。
  3. 相对冗余的指标域 (C2 & C4 消融)：移除人口 (C2, ARI=0.815) 或口粮 (C4, ARI=0.846) 对最终分组改变较小，它们在多维空间中提供了一定重叠信息。

省份稳定性分类解析：

  • 1. 强力绝对稳定核 (稳定实验数=6/6)
    包含上海、北京、天津、内蒙古、吉林、辽宁、黑龙江、西藏、宁夏、新疆、甘肃、贵州、青海、山东、广东、福建（上图中深色高亮省份）。这些省区在所有消融与等权配置试验中，聚类簇号始终不变，具有无可动摇的模板意义，是刻画各簇画像的骨干节点。
  • 2. 高稳定性省份 (稳定实验数=5/6)
    包含江苏、浙江、重庆、山西、海南（浅色部分）。这类省区具有极高稳定性，仅在某些特定指标消融（如山西在C1，苏、浙、渝在C3，海南在C5时）才发生轻微重组。
  • 3. 易漂移过渡地带 (稳定实验数=3-4/6)
    典型代表是河北、河南、江西、湖北。以豫、冀为例，因其身兼北方粮食主产带（类0）与人口稠密及消费水平（类3）的折中属性，在消融不同特征时会在类0与类3之间摇摆漂移。
1. 域内等权校正因子设计 build_experiment_matrix 
def build_experiment_matrix(
    scaled_df: pd.DataFrame,
    domains: list[str],
    direct_feature_equal: bool,
) -> tuple[np.ndarray, list[str]]:
    selected_cols = [col for domain in domains for col in DOMAIN_COLS[domain]]
    if direct_feature_equal:
        return scaled_df[selected_cols].to_numpy(dtype=float), selected_cols
    blocks = []
    for domain in domains:
        cols = DOMAIN_COLS[domain]
        # 对每一个域内的特征除以对应特征数目的平方根
        # 从而将每个域的期望方差之和修正为 1.0 (等额贡献)
        blocks.append(scaled_df[cols].to_numpy(dtype=float) / sqrt(len(cols)))
    return np.concatenate(blocks, axis=1), selected_cols
2. 贪心暴力枚举最优簇标签对齐 align_labels_to_baseline 
def align_labels_to_baseline(labels: np.ndarray, baseline: np.ndarray) -> tuple[np.ndarray, dict[int, int]]:
    best_overlap = -1
    best_mapping: dict[int, int] | None = None
    unique = sorted(np.unique(labels).tolist())
    # K=5, 共有 5! = 120 种排列组合，直接暴搜耗时微秒级
    for permutation in itertools.permutations(range(N_CLUSTERS)):
        mapping = dict(zip(unique, permutation, strict=True))
        mapped = np.array([mapping[int(label)] for label in labels], dtype=int)
        overlap = int((mapped == baseline).sum()) # 计算与主实验重合的省份数量
        if overlap > best_overlap:
            best_overlap = overlap
            best_mapping = mapping
    if best_mapping is None:
        raise ValueError("对照实验簇编号未能与 M0 对齐。")
    aligned = np.array([best_mapping[int(label)] for label in labels], dtype=int)
    return aligned, best_mapping

1

多源长表聚合与行政区划代码对齐

以中科院地理所的静态土地面积表的代码（海南校正为 460000）为基准，在导入时利用 Pandas 自动强制转换并映射对齐，校正过程均保留在 行政区划代码校正记录.csv 中。

2

非对称尺度平滑

人口密度最高与最低省份相差过千倍，可支配收入绝对趋势斜率同样分布偏斜。模型在提斜率前，先执行 log1p(x) = ln(1+x)，再做列 z-score 标准化。

3

严格的一致性复现超参数

所有 KMeans 聚类过程均显式绑定 random_state=42、n_init=80 以及 max_iter=300，从而杜绝局部极小值造成的簇标签漂移。

4

断言检查自动哨兵 (Checks Pipeline)

在执行输出保存前，会自动对 12 项检查项目进行真值检查。一旦出现 NaN 值、空簇或省份遗失，程序会抛出异常自动中断，保证了数据的生产级鲁棒性。
